# 这里是哪里?
这里是`StoneWeather`的开源库.

这是一个能在`Disocrd`运行的~~鸡气人~~机器人，目前支持`zh_CN`和`en_US`两种语言

由~~SB~~WE用`C#`语言编写。

## 指令
```
s!help //查看帮助
s!sw <城市名称> //查某个地方的天气
s!config Lang <语言> //对自己使用特定语言，只支持zh_CN和en_US两个值
```
